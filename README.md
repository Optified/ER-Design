# ER-Design

This is a report on the thought process of how to transition a spreadsheet of employees into a database by using ER model practices, and database integrity rules. 

The spreadsheet: Employees.xlsx is a random spreadsheet of information of individuals who are not real that was used as the reference for this report.
The report: ER Design.docx, is the report itself that goes over the thought process of creating an ER model that works for the use-case of the employees and storing payment and credential info.
